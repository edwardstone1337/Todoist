How it works

  • At minimum, only Task name is required.
  • If no project is entered, Todoist will use your Inbox.
  • URLs work best on mobile and when scanned from QR codes.
  • You can write this URL to an NFC tag using apps like NFC Tools.
  • If project name doesn't match exactly, Todoist may use Inbox instead.
